# Meeting Analyzer

A meeting analyzer built for Indian teams that speak Hinglish.

## What it does
- Records meeting audio from browser
- Transcribes using OpenAI Whisper
- Extracts summary and action items using LLaMA 3.3 via Groq API

## Why it's different
- Built for Hinglish — how Indian teams actually talk
- Works for informal meetings, not just formal corporate calls

## Tech stack
- Whisper, Groq API, LLaMA 3.3, Python
